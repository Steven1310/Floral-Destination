Steps to Execute Locally:
1)Download and Install Wamp Server
https://www.wampserver.com/en/
2)Open Wamp and place the repository inside the www directory of wamp
3)Start phpMyAdmin
Note:Default username is "root" paswword is empty and database as MySQL
4)Login with default Credentials
5)Crate new database named "flower"
6)Import and run the flower.sql file
7)Make sure that Wamp is up and running and then navigate to http://localhost/Floral_Website/
